# Client Server Core

Распределенное клиент-серверное приложение для управления коллекцией объектов (Movie). Проект демонстрирует работу с сетевыми протоколами, многопоточностью, сериализацией и проектированием объектных моделей.

---

##  Ключевые возможности

*   **Многомодульная архитектура:** Разделение на `Client`, `Server` и `Common`.
*   **Гибридный интерфейс:** Поддержка графического интерфейса (GUI) и консольного клиента (CLI).
*   **Сетевой протокол:** Взаимодействие по TCP с кастомной сериализацией (Jackson).
*   **Валидация данных:** Двухуровневая проверка данных (на клиенте и на сервере).
*   **Логирование:** Использование Log4j2 для аудита событий и отслеживания ошибок.

---

## Предварительная настройка (.env)

**Важно:** Перед запуском необходимо создать файл `.env` в корневом каталоге проекта. Без него приложение будет использовать параметры по умолчанию (могут не подойти для БД).

Создайте файл `.env` и заполните его:
```env
# Network
PORT_CLIENT=8080
PORT_SERVER=9090
HOST=localhost
CLIENT_TIMEOUT=5000

# Database
DB_URL=jdbc:postgresql://localhost:5432/lab7
DB_USER=postgres
DB_PASSWORD=password
```

---

**GUI frameworks** Есть варианты выбора JavaFX / Swing. Дефолтным стоит JavaFX. Его можно поменять в client/Main.java

## Технологический стек

*   **Язык:** Java 17+
*   **Сборка:** Maven (Multi-module)
*   **Библиотеки:** 
    *   `Log4j2` — логирование.
    *   `Jackson` — JSON парсинг и сетевой протокол.
    *   `PostgreSQL` — хранение коллекции.
*   **Документация:** JavaDoc (доступна через GitHub Pages).

## Запуск проекта

### 1. Сборка всех модулей
```bash
mvn clean package
```

### 2. Способ А: Запуск через скомпилированные JAR (Рекомендуется)
Запускать строго из корня проекта, чтобы подтянулся `.env`:

**Сервер:**
```bash
java -jar server/target/server-1.0-jar-with-dependencies.jar
```
**Клиент:**
```bash
java -jar client/target/client-1.0-jar-with-dependencies.jar
```

### 3. Способ Б: Запуск через Maven
Убедитесь, что вы находитесь в корневой директории:

**Сервер:**
```bash
mvn exec:java -pl server -Dexec.mainClass="server.Main"
```
**Клиент:**
```bash
mvn exec:java -pl client -Dexec.mainClass="client.Main"
```
